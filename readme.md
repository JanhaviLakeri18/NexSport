# NexSport - Sports E-Commerce Website

NexSport is a full-stack sports e-commerce web application built using React, TypeScript, Spring Boot, and MySQL.

## Features

- Modern responsive UI
- Product catalog
- Product details page
- Add to cart
- Basket management
- Login/Register UI
- Contact page
- Search and filters
- Redux state management
- MySQL database integration

## Tech Stack

### Frontend
- React
- TypeScript
- Material UI
- Redux Toolkit
- React Router

### Backend
- Spring Boot
- Java
- MySQL

## Run Frontend

```bash
cd client
npm install
npm run dev
```

##Run Backend

```bash
./mvnw spring-boot:run
```
